# TaskFlow

A modern task management web app built with HTML, CSS and Vanilla JavaScript.

---

## Features

- Add tasks
- Delete tasks
- Mark tasks as complete
- Search tasks
- Filter by All, Completed and Pending
- Local storage support
- Dynamic empty states
- Responsive dark UI
